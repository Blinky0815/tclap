tclap -- Templatized C++ Command Line Parser Library

This is a small library that provides a simple interface for defining and accessing command line arguments. It was intially inspired by the user friendly CLAP libary. The difference is that this library is templatized so the argument class is type independent which avoids identical-except-for-type objects like IntArg, FloatArg, StringArg, etc. Someday I'll bother to figure out the POSIX and/or GNU styles and make sure that all manner of arguments are supported. For now its my interpretation of what most argument situations are.
