tclap -- Templatized C++ Command Line Parser Library

This is a small library that provides a simple interface for defining and accessing command line arguments. It was intially inspired by the user friendly CLAP libary. The difference is that this library is templatized so the argument class is type independent which avoids identical-except-for-type objects like IntArg, FloatArg, StringArg, etc. While the library is not strictly compliant with the GNU or POSIX standards, it is getting close.
